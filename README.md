# GigFlow - Freelancing Platform

A real-time freelancing platform where clients can post gigs and freelancers can bid on them.

## Features
- User authentication (Client/Freelancer)
- Post gigs with details and budget
- Real-time bidding system
- Hire freelancers from bids
- Dashboard for both roles

## Tech Stack
- **Backend**: Node.js, Express, MongoDB, Socket.io
- **Frontend**: React, Redux Toolkit, Tailwind CSS
- **Database**: MongoDB
- **Real-time**: Socket.io
